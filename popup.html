Do. Settings.

If you do not wish to use your own OpenAI API key, then you can use the /do endpoint for free.

Just leave the key field empty. In return, on occasion we will respond with something that resembles an ad, you can just ignore this or use the backtick (`) and try again.

` (backtick) after text generation undo's everything and returns to your command. You can try again if you want.

{text} contains all the text in the textarea or input field. Use it like so: /do summarize the following {text}
